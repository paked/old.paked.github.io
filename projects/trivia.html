While at NCSS 2015 I worked with my team, group 4 to build a Trivia website named quizzi. Users on the website could participate in trivia games and add questions for other users to complete. Personally I, along with one other student built the Object Relational Mapping (ORM) system. The ORM is used to abstract the database into a set of models which can be used by the other people in our group. Towards the end of the project I became responsible for developing the actual "Game" system that allowed users to complete matches and be assigned scores.
