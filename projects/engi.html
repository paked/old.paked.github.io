Engi is a game engine, written in Go following the Entity Component System (ECS) paradigm. It cross compiles to Linux, Mac OSX, Windows and the web. The ECS design pattern is known from engines like Source and Unity.

Originally, Engi was a fork of github.com/ajhager/engi. Ajhager had made leaps and bounds from a rendering perspective for a Go game engine. Unfortunately the library was becoming stagnant and not heading in a direction I liked. I'm maintaining a fork of it which is aimed at making it more of a game engine than a multimedia library.
